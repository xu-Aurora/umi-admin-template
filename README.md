# umi project

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```


```bash
注意点: 
使用umi-hooks:
  1、umi框架内置的useRequest和单独报下载'@umijs/hooks'里useRequest有些不同，
umi 里面的 useRequest 要求相应的结果必须是这样 {data: {xxx}} ，要用 data 包一下
```
